# MoneyPop: CirclePop 🔵⬆️

**MoneyPop: CirclePop**은 아래에서 위로 원형 도형을 발사하여 동일한 단계의 도형을 합치고, 목표 도형을 완성하여 점수를 얻는 **역중력 머지(Merge) 퍼즐 게임**입니다. Next.js와 Matter.js 물리 엔진을 활용해 웹 기반의 미니앱 환경에서 부드러운 타격감과 이펙트를 제공하도록 개발되었습니다.

<br/>

## 🎮 게임 방식 (How to Play)
1. **역중력 물리 엔진**: 게임 내 중력은 위쪽으로 향하여, 모든 도형은 상단으로 끌려 올라갑니다.
2. **발사 및 조작**: 하단의 발사 레일에서 다음 스폰될 도형을 좌우로 드래그 하여 위치를 정한 뒤, 손을 놓아 수직으로 쏘아 올립니다.
3. **병합(Merge)**: 쏘아 올린 도형이 맵에 있는 동일한 단계의 도형과 만나면 한 단계 더 큰 위의 도형으로 합쳐집니다. (예: 1단계 + 1단계 = 2단계)
4. **목표 달성**: 상단에 제시된 **타겟 도형(Target)** 단계를 맵에서 만들어내면, 즉시 제거되면서 큰 점수를 획득합니다!
5. **게임 오버**: 다음 도형을 발사할 수 있는 빈 공간(x 좌표)이 아무데도 남아있지 않으면 게임이 종료됩니다.

<br/>

## ✨ 주요 특징
* **가벼운 전략 기반 판단**: 순발력보다 현재 필드를 보고 어느 위치에 도형을 밀어 올릴지 전략적으로 배치하는 재미를 중시합니다.
* **연쇄 합체(Combo Merging)**: 한 번의 발사로 여러 번 연속해서 도형들이 병합되는 연쇄 작용이 일어날 수 있습니다. 
* **심플하고 직관적인 UI/UX**: 유저의 몰입을 방해하지 않는 깔끔한 다크 모드 기반 뷰와 부드러운 애니메이션 효과를 도입했습니다.

<br/>

## 🛠 기술 스택 (Tech Stack)
* **프레임워크**: Next.js (App Router), React
* **스타일링**: Tailwind CSS v4, CSS Variables
* **물리 엔진**: Matter.js (자유 좌표 기반 물리 연산, 충돌 판정, 역중력 적용)

<br/>

## 🚀 로컬 실행 방법 (Running Locally)

터미널을 열고 본 프로젝트 폴더로 이동한 후, 다음 명령어들을 실행하세요.

```bash
# 1. 의존성 설치
npm install

# 2. 로컬 개발 서버 실행
npm run dev
```

이후 브라우저에서 `http://localhost:3000`에 접속하여 직접 플레이 해보실 수 있습니다.
